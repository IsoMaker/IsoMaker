Detailed Description

Main application UI controller class.

The MainUI class serves as the central controller for the IsoMaker game engine editor. It manages the main application window, camera, and orchestrates communication between the UI system and various editor components. This class is responsible for the main application loop and high-level coordination of all editor subsystems.

Key responsibilities:

  • Main application lifecycle management
  • Window and camera management
  • Coordination between UI and editor components
  • Input handling coordination
  • Main rendering loop
